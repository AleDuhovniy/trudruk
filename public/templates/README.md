# Шаблоны документов

Сервис ожидает готовые Word-файлы в этой папке.

Рекомендуемая структура:

- `public/templates/common/`
- `public/templates/tardiness/`
- `public/templates/absence/`
- `public/templates/nonperformance/`
- `public/templates/regulation/`
- `public/templates/intoxication/`

Имена файлов можно оставить такими, какие указаны в `src/data/scenarios.ts`.
Если файл еще не подключен, интерфейс покажет, что шаблон будет добавлен позже.
