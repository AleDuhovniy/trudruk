# trud

## Рекомендованный хостинг

Для этого проекта лучше всего подходит GitVerse Pages:

- это российский сервис;
- он бесплатный для статических сайтов;
- вашему проекту не нужен сервер, потому что здесь React + Vite и итогом сборки является папка `dist`.

Важно:

- репозиторий для GitVerse Pages сейчас должен быть публичным;
- пользовательские домены в текущей версии GitVerse Pages пока не поддерживаются;
- адрес сайта будет вида `https://<username>.gitverse.site/trud/`.

## Что уже настроено в проекте

В проект добавлены:

- относительные пути для корректной работы в подпапке GitVerse Pages;
- workflow для автопубликации в `.gitverse/workflows/publish-pages.yml`.

## Как опубликовать

1. Создайте аккаунт на GitVerse.
2. Создайте новый публичный репозиторий `trud`.
3. Загрузите туда этот проект.
4. В GitVerse откройте `Настройки -> CI/CD` и включите CI/CD.
5. Затем откройте `Настройки -> Страницы`.
6. Включите Pages и выберите источник `Воркфлоу`.
7. Сделайте push в ветку `main` или запустите workflow вручную.
8. После успешной сборки сайт появится по адресу `https://<username>.gitverse.site/trud/`.

## Локальная проверка

```bash
npm ci
npm run build
```

Если понадобится вариант без публичного репозитория, следующим по удобству я бы рассматривал Beget Free Hosting: туда можно просто загружать содержимое папки `dist` после сборки.
